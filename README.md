# Akıllı Otopark ve Plaka Tanıma Projesi

Bu proje, otomata teorisi kapsamında geliştirilen akıllı otopark ve plaka tanıma sistemidir. Projede araç plakası ve otopark durumu gibi girdiler alınarak sistemin durum geçişleri takip edilir.

## Proje Amacı

Bu projenin amacı, sonlu otomata mantığını gerçek hayata yakın bir senaryo üzerinde uygulamaktır. Akıllı otopark sisteminde araç girişleri, plaka kontrolü, otopark doluluk durumu ve geçiş kararları otomata yapısı ile modellenmiştir.

## Proje İçeriği

- Plaka tanıma mantığı
- Araç giriş kontrolü
- Otopark doluluk kontrolü
- Durum geçişleri
- Kabul ve ret durumları
- Kullanıcıdan veri alma
- Arayüz üzerinden sistemin çalıştırılması

## Kullanılan Teknolojiler

- Python
- Tkinter
- Otomata Teorisi
- Git
- GitHub

## Sistem Mantığı

Sistem, kullanıcıdan alınan bilgilere göre farklı durumlara geçiş yapar. Araç plakası uygun formatta ise ve otoparkta boş yer varsa araç kabul edilir. Plaka hatalıysa veya otopark doluysa sistem aracı reddeder.

## Durumlar

- Başlangıç durumu
- Plaka kontrol durumu
- Otopark kapasite kontrol durumu
- Kabul durumu
- Ret durumu

1. Projeyi bilgisayarınıza indirin.
2. Python kurulu olduğundan emin olun.
3. Proje klasörünü VS Code ile açın.
4. Terminalden aşağıdaki komutu çalıştırın:
